Trivia Quiz App (JavaFX)
A simple JavaFX Single-Page Trivia Quiz App that uses the Open Trivia DB API.

Requirements
Java 17+
Maven
(Optional) Scene Builder to edit FXML
Run
mvn clean javafx:run
Features
Choose number of questions, category, difficulty, and type
Fetches questions from Open Trivia DB
Shows questions one by one, indicates correct/incorrect, and shows final score
